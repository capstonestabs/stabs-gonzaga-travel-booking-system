# Features

## Guest / Visitor Features

- Browse the homepage, destinations, feedback, and about pages
- View destination details, cover images, and gallery images
- View service packages and pricing
- View destination location and public contact details
- View a read-only availability calendar for online-bookable services
- View walk-in destinations as contact-only listings
- Cannot book, reserve, or proceed to payment without an account

## Tourist Features

- Create an account
- Sign in to the system
- Confirm email before first sign-in
- Recover password by email
- Browse destinations and select services
- Use a visual calendar to check service availability
- Continue to checkout and pay through PayMongo GCash
- View booking status after payment
- Access a tourist dashboard
- View active bookings
- View booking history
- View a personal booking calendar
- Access a ticket wallet
- Open and download booking tickets
- Cancel pending reservations before completion
- Clear completed, expired, or cancelled booking history

## Staff Features

- Manage one assigned destination
- Update destination summary, description, and category
- Update public destination contact email and phone
- Upload and manage cover photo and gallery images
- Create, edit, activate, deactivate, and delete service packages
- Add optional service photos
- Set service pricing
- Set daily capacity
- Set service availability start and end dates
- Manage service closure dates through a visual calendar
- View destination bookings
- Mark eligible bookings as completed
- Delete bookings only when allowed by booking state

## Admin Features

- View admin dashboard metrics
- Create staff accounts
- Reset staff passwords
- Manage tourist accounts
- Manage staff accounts
- Delete staff or tourist access from active use while preserving historical data
- Monitor active payout records
- Record payout settlement
- Manage payout history
- Delete individual payout history rows
- Delete payout history per destination
- Delete all payout history

## Booking and Payment Features

- Service-based reservation flow
- Temporary slot hold before payment
- Capacity tracking by guest count
- Confirmation only after successful payment
- Ticket generation after confirmation
- GCash-only checkout through PayMongo
- Failed or expired payment releases capacity

## Walk-In Features

- Destination can be marked as walk-in
- Walk-in listings still show services and prices
- Walk-in listings show contact details and address
- No online booking or payment flow for walk-in destinations
- Walk-in inquiries do not reduce online slot availability

## Account and Security Features

- Role-based access control
- Tourist-only forgot-password email flow
- Staff password reset handled by admin
- In-app confirmation dialogs instead of browser popups
- Protected private routes through middleware and server-side role checks

## Media and UI Features

- Responsive public and dashboard pages
- Mobile-friendly layouts
- Clickable gallery image enlargement
- Destination cover hero layout
- Visual booking and management calendars
- Downloadable ticket view

## Not Included

- Refund workflow
- Cancellation after successful payment
- Admin-created offline walk-in bookings
- Airline or international travel services
- Multi-gateway payment support beyond current PayMongo GCash flow
